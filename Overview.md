JobTrack AI - Project Overview
1. Project Objective
The objective of this project is to design and develop a functional digital application that streamlines the internship and job search process for students. By following the Full-Code Development Track, this project implements a complete lifecycle from requirements analysis to a final working product.

2. Problem Statement & Target Users

User Problem: Students often struggle to track dozens of job applications across different platforms, leading to missed deadlines and generic, ineffective cover letters.


Target Users: University students and recent graduates actively seeking internships or entry-level roles.

3. Core Features (MVP Scope)
To exceed the standard project requirements, JobTrack AI includes advanced AI and automation features:


Intelligent Kanban Board: A drag-and-drop interface with columns for Wishlist, Applied, Interviewing, Offer, and Rejected .


AI Match Scoring: A Django service that compares a user's CV against a job description using Google Vertex AI to provide a 0-100% suitability score .


AI Cover Letter Architect: Automated drafting of personalized cover letters tailored to specific job requirements.


Google Calendar Automation: Automatic interview scheduling and reminders when an application moves to the "Interviewing" stage .

4. Technical Stack
This project utilizes the Full-Code Development Track stack :


Frontend: React for a responsive user interface.


Backend: Django (Python) for core logic and API management .


Database: PostgreSQL for relational data storage.

AI Engine: Google Cloud Vertex AI (Gemini 1.5) for resume analysis and content generation.


Version Control: GitHub for collaboration and commit history tracking .
